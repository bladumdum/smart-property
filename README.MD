# Smart Property

Aplikasi prediksi harga rumah menggunakan Random Forest Regressor.
Link Website [Smart-Property](https://bladumdum.github.io/projek-akhir/)

## Teknologi

### Frontend

- React
- Vite
- Tailwind CSS

### Backend

- Python
- Flask
- MySQL

### Machine Learning

- Random Forest Regressor
- Scikit-learn

## Fitur

- Prediksi harga rumah
- Penyimpanan riwayat prediksi
- Hapus riwayat prediksi
- Landing page informatif

## Struktur Project

backend/
smart-property/

## Cara Menjalankan

### Backend

pip install -r requirements.txt
python app.py

### Frontend

cd smart-property
npm install
npm run dev
